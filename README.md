# Aplikasi Pengadaan Barang (SPMS)

Aplikasi berbasis web dan desktop untuk manajemen pengadaan barang sekolah (Sistem Pendaftaran Barang). Memungkinkan Inventaris untuk mengajukan barang, Direktur untuk menyetujui, dan Admin untuk membelikan barang.

## Fitur Utama
- **Multi-Role Login**: Akses khusus untuk Inventaris, Direktur, dan Admin.
- **Sistem Persetujuan**: Alur kerja transparan untuk pengajuan dan persetujuan barang.
- **Notifikasi Lonceng (In-App)**: Pemberitahuan cerdas dan real-time sesuai peran pengguna.
- **Statistik Dashboard**: Visualisasi data penggunaan anggaran dan status barang.
- **Aplikasi Desktop**: Tersedia dalam bentuk `.exe` untuk kemudahan akses di Windows.

## Teknologi
- HTML5, CSS3, Vanilla JavaScript
- SheetDB (Backend Database)
- Electron & Nativefier (Desktop Wrapper)
